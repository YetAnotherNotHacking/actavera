# ActaVera
An open public text sharing site built for Tor. Designed for people to publically post ciphertext to a public pool that people can access by id and decrypt locally with a known passphrase.

## Public links:
Clearnet: [link](https://actavera.silverflag.net)
Onion service: await

## Installation
1. Clone the git repo
2. Run `python3 setup.py`
3. Once it completes, note the onion addres
4. Run `python3 app.py` in a screen or nohup environment.
